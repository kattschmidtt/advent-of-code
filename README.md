## Run Locally

Clone the project

```bash
  git clone git@github.com:kattschmidtt/advent-of-code.git
```

Go to the project directory

```bash
  cd advent-of-code
```

Install dependencies

```bash
  npm i -D typescript ts-node
```

Run the code (replace x with date number)

```bash
   cd Dayx
   npx tsc .\dayx.ts
   node .\dayx.js
```
